## 1. 系统要求

### 1.1 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 4 核 | 8 核以上 |
| 内存 | 8 GB | 16 GB |
| 磁盘 | 10 GB 可用空间 | 50 GB 以上（视频存储） |
| GPU（可选） | - | NVIDIA GPU >= 6 GB VRAM（加速 Whisper 转写） |

**说明：** 不使用 GPU 也可以运行，但语音转写速度会慢 3-5 倍。

### 1.2 软件要求

| 软件 | 版本要求 | 用途 |
|------|---------|------|
| Python | >= 3.10, < 3.13 | 运行环境 |
| ffmpeg | >= 6.0 | 音视频处理 |
| pip | >= 23.0 | 包管理 |

## 2. 环境安装

### 2.1 Python 环境

```bash
conda create -n videonote python=3.11
conda activate videonote
# 或 venv
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate
```

### 2.2 安装 ffmpeg

**Windows：** `winget install ffmpeg` 或手动下载后添加到系统 PATH

**macOS：** `brew install ffmpeg`

**Linux：** `sudo apt update && sudo apt install ffmpeg`

**验证：** `ffmpeg -version`

### 2.3 安装 Python 依赖

```bash
pip install -r requirements.txt
# 或手动安装
pip install fastapi uvicorn[standard] sqlalchemy alembic yt-dlp
pip install httpx pydantic pydantic-settings loguru chromadb faster-whisper
```

**requirements.txt：**

```txt
fastapi>=0.104.0
uvicorn[standard]>=0.24.0
sqlalchemy>=2.0.0
alembic>=1.13.0
yt-dlp>=2024.0.0
httpx>=0.26.0
pydantic>=2.0.0
pydantic-settings>=2.0.0
loguru>=0.7.0
chromadb>=0.5.0
faster-whisper>=1.0.0
```

## 3. 配置 API Key

| 服务 | 用途 | 获取方式 |
|------|------|---------|
| 通义千问 | 笔记生成/问答 | 阿里云百炼平台（新用户免费额度） |
| DeepSeek | 备选 LLM | DeepSeek 开放平台 |
| 必剪 API | 转写备选 | 必剪开放平台 |

**.env 文件：**

```env
TONGYI_API_KEY=sk-xxxxxxxxxxxxxxxx
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
LLM_PROVIDER=tongyi
WHISPER_MODEL_SIZE=medium
WHISPER_DEVICE=auto
```

## 4. 运行项目

### 4.1 初始化

```bash
mkdir -p data/videos data/chromadb logs
alembic upgrade head
```

### 4.2 启动

```bash
# 开发模式（热重载）
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
# 生产模式
uvicorn app.main:app --host 0.0.0.0 --port 8000 --workers 1
```

### 4.3 验证

```bash
curl http://localhost:8000/api/v1/system/health
# 浏览器打开 http://localhost:8000/docs
```

### 4.4 快速端到端验证

```bash
curl -X POST http://localhost:8000/api/v1/videos/parse -H "Content-Type: application/json" -d '{"url":"https://www.bilibili.com/video/BV1xx411c7mD"}'
curl -X POST http://localhost:8000/api/v1/videos/process -H "Content-Type: application/json" -d '{"url":"https://www.bilibili.com/video/BV1xx411c7mD","quality":"720p"}'
curl http://localhost:8000/api/v1/notes/{video_id}
```

## 5. 常见问题

| 问题 | 解决 |
|------|------|
| ffmpeg 找不到 | 确认 ffmpeg 在系统 PATH 中 |
| GPU 显存不足 | model_size 改为 small 或 device: cpu |
| 下载失败 | 更新 yt-dlp，请求间隔 >= 10s |
| API 调用失败 | 检查 API Key 和网络连接 |
| 数据库打不开 | 确认 data/ 目录存在且有写权限 |

