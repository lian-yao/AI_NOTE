## 2. 后端目录（app/）

### 2.1 目录总览

```
app/
├── api/v1/       # FastAPI 路由层，5 组 RESTful 接口
├── core/         # 基础设施：配置、错误处理、日志
├── llm/          # LLM 统一客户端（通义千问 + DeepSeek）
├── models/       # SQLAlchemy ORM 数据模型（5 个表）
├── note/         # 笔记生成器（LLM prompt + Markdown 解析）
├── pipeline/     # 流水线编排 + 事件总线
├── processor/    # 视频处理：解析/下载/音频提取
├── qa/           # 智能问答引擎
├── retriever/    # 混合检索器（向量 + 关键词 + RRF）
├── schemas/      # Pydantic 请求/响应模型
├── store/        # 向量化存储（切片 + embedding + ChromaDB）
└── transcriber/  # 语音转写（Whisper + 必剪 API）
```

### 2.2 各包说明

| 目录 | 职责 | 对应文档中的模块 | 输入 | 输出 |
|------|------|----------------|------|------|
| api/v1/ | HTTP 接口路由，参数校验，响应组装 | API 接口文档 | HTTP 请求 | HTTP 响应 |
| core/ | 配置加载、全局异常处理器、日志初始化 | 技术方案 6/7 节 | 环境变量/config.yaml | 配置对象 |
| llm/ | 统一封装 LLM chat 和 stream_chat 调用 | Prompt 工程文档 4 节 | messages 列表 | 文本/流式生成 |
| models/ | SQLAlchemy ORM 声明式映射 | 数据库设计文档 7 节 | - | ORM 模型类 |
| note/ | 构建 prompt 调 LLM 生成笔记，解析输出 | 技术方案 3.3 节 | 转写文本 + 元数据 | StageResult |
| pipeline/ | 编排各模块的执行顺序、并发控制、事件推送 | 流水线任务编排设计 | TaskContext | PipelineEvent |
| processor/ | yt-dlp 下载、ffmpeg 提取音频、URL 解析 | 技术方案 3.1 节 | 视频 URL | 视频/音频文件 |
| qa/ | 检索增强问答上下文构建 + LLM 流式回答 | 技术方案 3.5 节 | 问题 + video_id | 流式回答 + 引用 |
| retriever/ | 向量相似度 + 关键词 + RRF 排序融合 | 技术方案 3.5 节 | 问题文本 | 排序后的切片列表 |
| schemas/ | Pydantic 模型定义，API 请求/响应序列化 | API 接口文档 8 节 | - | Pydantic BaseModel |
| store/ | 按标题语义切片、text-embedding-v3 向量化、ChromaDB 读写 | 技术方案 3.4 节 | note.md | ChromaDB 向量 |
| transcriber/ | 策略模式封装 Faster-Whisper 和必剪 API | 技术方案 3.2 节 | audio.wav | transcription.json |

### 2.3 模块依赖关系

```
api/v1/  (路由层，无业务逻辑)
  ├── 依赖 pipeline/ 提交视频处理
  ├── 依赖 store/ 查询向量
  └── 依赖 qa/ 处理问答

pipeline/  (编排层，调度各模块)
  ├── processor/  下载视频、提取音频
  ├── transcriber/ 转写音频
  ├── note/       生成笔记
  └── store/      向量化存储

qa/  (问答层)
  ├── retriever/  检索相关片段
  ├── llm/        调用大模型生成回答
  └── store/      查询向量库
```

## 3. 前端目录（frontend/）

### 3.1 目录总览

```
frontend/
├── public/               # 静态资源
├── src/
│   ├── api/              # HTTP 请求封装 + Mock 数据
│   │   └── mock/         # 后端未就绪时的模拟数据
│   ├── components/       # Vue 组件
│   │   ├── common/       # 通用组件（10 个）
│   │   ├── video/        # 视频相关组件（5 个）
│   │   ├── note/         # 笔记相关组件（4 个）
│   │   ├── qa/           # 问答相关组件（5 个）
│   │   └── settings/     # 设置页组件（4 个）
│   ├── router/           # Vue Router 路由定义
│   ├── stores/           # Pinia 状态管理（5 个 Store）
│   ├── styles/           # 全局样式变量
│   ├── types/            # TypeScript 类型定义
│   └── views/            # 5 个页面组件
├── index.html
├── vite.config.ts
├── tsconfig.json
└── package.json
```

### 3.2 各目录说明

| 目录 | 内容 | 组件数量 |
|------|------|---------|
| api/ | Axios 实例 + 5 组 API 封装（videos/notes/qa/tasks/system）+ Mock 数据 | 5 个接口模块 |
| components/common/ | AppLayout、SidebarNav、TopBar、StatusBadge、ProgressBar、VideoCard、ConfirmDialog、EmptyState 等 | 10 个 |
| components/video/ | VideoLinkInput、VideoMetaCard、VideoTable、ProcessOptions、TaskProgressList | 5 个 |
| components/note/ | NoteViewer、NoteSection、NoteRawDownload、KeywordTags | 4 个 |
| components/qa/ | ChatPanel、ChatMessage、ChatInput、ReferenceCard、ScopeSelector | 5 个 |
| components/settings/ | ApiKeyInput、ModelSelector、StorageManager、SystemStats | 4 个 |
| router/ | 5 条路由定义（工作台/视频列表/视频详情/问答/设置）| 1 个模块 |
| stores/ | videoStore、noteStore、qaStore、taskStore、systemStore | 5 个 Store |
| views/ | DashboardView、VideoListView、VideoDetailView、QAView、SettingsView | 5 个页面 |

### 3.3 页面路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 工作台 | / | 链接输入、统计概览、最近处理 |
| 视频列表 | /videos | 搜索、筛选、排序 |
| 视频详情 | /videos/:videoId | 元数据 + 笔记 + 问答 |
| 问答 | /qa | 跨/单视频问答 |
| 设置 | /settings | API Key/模型/存储 |

## 4. 数据目录（data/）

```
data/
├── chromadb/    # ChromaDB 向量数据库持久化文件
├── logs/        # 运行日志（按日期轮转）
└── videos/      # 视频处理中间产物
    └── {video_id}/
        ├── meta.json           # 解析阶段 -> 元数据
        ├── video.mp4           # 下载阶段 -> 视频文件
        ├── audio.wav           # 提取阶段 -> 音频（16kHz mono）
        ├── transcription.json  # 转写阶段 -> 带时间戳文本
        ├── transcription.srt   # 转写阶段 -> 字幕文件
        └── note.md             # 生成阶段 -> 结构化笔记
```
