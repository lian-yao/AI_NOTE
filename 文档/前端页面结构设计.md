## 5. 状态规范

| 状态 | 表现 | 说明 |
|------|------|------|
| loading | 骨架屏 / 加载图标 | 首次加载 |
| empty | 插画 + 提示 + 行动按钮 | 列表为空 |
| error | 错误提示 + 重试按钮 | 请求失败 |
| success | 正常渲染数据 | 默认状态 |

```vue
<EmptyState
  v-if="!loading && videos.length === 0"
  title="还没有处理过视频"
  description="从工作台粘贴一个 B 站链接开始吧"
>
  <template #action>
    <router-link to="/">去处理第一个视频</router-link>
  </template>
</EmptyState>
```
